Projects

Secure Access Migration for the schools

We’re setting up a way for schools to manage who can access the network, and what they’re able to do when they’re on it. (We’re using a solution called Cloudpath by Ruckus for this.) It means devices signing onto your network will get unique login details, which identify which part of the network they can access and what they’ll be able to do when they’re on it. When the device is signed into the network for the first time, it will be recognized from then on - meaning you only need to log on once as the user’s device remembers it. (This is also referred to as ‘device onboarding’).
